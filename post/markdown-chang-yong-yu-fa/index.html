Markdown常用语法

2021-05-19

就是总结一下常用的语法
参考链接简书

标题

文字前面带几个#号，共有h1-h6六个级别
h1下面会有一条横线

列表

无序列表：文字前面加 + - * 三者中任意一个
有序列表：文字前面加 数字和点号
列表嵌套，使用tab 或 空格 缩进控制列表的层数

###引用说明区块
在文字前面加>即可，可以多加几个>嵌套

###引用说明区块
在文字前面加>即可，可以多加几个>嵌套

###代码块
单行代码前后分别用` 括起来
多行代码前后分别用``` 括起来
注意是左上角~标志

###链接
用[文字](链接 "描述" )
或者[文字]：链接 "描述"

###分割线
由于至少连续3个 * 或 - 或 _ 符号表示

###文字
加粗：文字前后分别用*或_括起来
倾斜：文字前后分别用**或__括起来
删除：文字前后分别用~~括起来

###表格

123|234|345
:-|:-:|-:
abc|bcd|cde
abc|bcd|cde
abc|bcd|cde
123 234 345
abc bcd cde
abc bcd cde
abc bcd cde

三列分别为左对齐，居中，右对齐
